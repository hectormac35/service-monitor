# Service Monitor – Frontend

Frontend del proyecto **Service Monitor**, una aplicación de monitorización de
servicios HTTP con enfoque DevOps / Observabilidad.

## Tecnologías
- Next.js
- TypeScript
- React Query
- Tailwind CSS
- Recharts

## Funcionalidad
- Dashboard de servicios monitorizados
- Visualización de estado UP / DOWN
- Gráficas de latencia y uptime
- Vista de detalle por servicio

⚠️ La documentación completa del proyecto se encuentra en el README principal
en la raíz del repositorio.
